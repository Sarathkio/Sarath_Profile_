# Sarathkumar Created By

This is a responsive website built using **HTML**, **CSS**, **Bootstrap**, and **JavaScript**. The website features a modern and sleek design, focusing on clean user interfaces and mobile responsiveness.

## Features
- **Responsive Design**: Uses Bootstrap grid system for responsiveness across devices.
- **Modern UI**: Sleek interface built with Bootstrap components.
- **Interactive Elements**: JavaScript for dynamic user interactions.
- **Content Sections**: Includes about, services, contact, and more.

## Technologies Used
- **HTML5**: Structure of the web page.
- **CSS3**: Styling and animations.
- **Bootstrap 5**: Responsive grid and pre-built components.
- **JavaScript**: Interactive and dynamic features.

## Installation

1. Clone this repository:
    ```bash
    git clone https://github.com/Sarathkio
    ```
2. Navigate to the project directory:
    ```bash
    cd sarathkumar-website
    ```
3. Open `index.html` in your browser.

## File Structure

- `index.html`: The main HTML file that includes the structure.
- `style.css`: Custom styles for the website.
- `script.js`: JavaScript for added interactivity.
